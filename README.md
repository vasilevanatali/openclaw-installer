# Установка OpenClaw «для ленивых» 🦞

Передаваемый скилл для **Claude Code** или **Codex**: отдаёшь его своему ИИ-ассистенту —
и он ведёт тебя по шагам от нуля до **своего ИИ-агента в Telegram**, который знает только
твои данные, работает 24/7 и ничего не сливает наружу.

Ты — «ленивый»: только присылаешь то, что просят (доступы к серверу, токен бота, код подписки,
скрин), и подтверждаешь шаги. Всю техническую работу (подключение, установка, настройка)
ассистент делает сам и проверяет каждый шаг.

## Что нужно

- Зарубежный сервер (VPS) ~300–500 ₽/мес
- Подписка **ChatGPT Plus** _или_ OpenAI **API-ключ** (это «мозг» бота)
- Аккаунт Telegram
- Claude Code или Codex на компьютере (это твой ассистент)

⏱ ~40 минут.

## Как использовать

### Вариант 1 — Claude Code

1. Скачай этот репозиторий (`git clone …`) или просто папку.
2. Открой её в Claude Code и скажи: **«Установи мне OpenClaw по этому гайду»**
   (или положи `SKILL.md` в свои скиллы — Claude подхватит его сам, когда попросишь поставить агента).
3. Дальше отвечай на вопросы ассистента — он ведёт по шагам.

### Вариант 2 — Codex

1. Открой `SKILL.md`, скопируй его содержимое.
2. Вставь как первое сообщение/инструкцию в Codex и напиши: **«Будь установщиком по этому гайду, начинай с Шага 0»**.
3. Дальше отвечай на вопросы — он ведёт по шагам.

## Шаги, по которым тебя проведут

1. Арендовать сервер
2. Подключиться к серверу (ассистент сам)
3. Поставить OpenClaw (ассистент сам)
4. Создать бота в Telegram, забрать токен
5. Подключить мозг (подписка ChatGPT или API-ключ)
6. Запустить — **бот отвечает** 🎉
7. Дать характер и базу знаний
8. Память `/learn` — готово

На каждом шаге ассистент проверяет, что всё реально сделано, и только потом идёт дальше.

## Безопасность

Токены, пароли и ключи ассистент кладёт только в нужные файлы на твоём сервере и не светит
их в чате. Не присылай свои секреты в публичные чаты. OpenClaw ставится только на зарубежный
сервер (не на ноут и не на сервер в РФ — иначе ИИ-модели не работают).
